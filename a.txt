HTML (HyperText Markup Language) เป็นภาษามาร์กอัปที่ใช้เพื่อสร้างโครงสร้างและเนื้อหาของหน้าเว็บมาตรฐาน มีองค์ประกอบหลายองค์ประกอบที่ใช้กันอย่างแพร่หลาย นี่คือรายการของบางองค์ประกอบ HTML ที่ใช้บ่อย:

1. <html>: แทนส่วนต้นของเอกสาร HTML
2. <head>: มีข้อมูลเมตาและข้อมูลอื่น ๆ เกี่ยวกับเอกสาร
3. <title>: ระบุชื่อเรื่องของเอกสารที่แสดงในแถบหัวเรื่องหรือแท็บของเบราว์เซอร์
4. <body>: แทนเนื้อหาของเอกสาร HTML
5. <h1> ถึง <h6>: กำหนดระดับของหัวเรื่องจากสำคัญที่สุด (h1) ถึงน้อยที่สุด (h6)
6. <p>: กำหนดย่อหน้า
7. <a>: สร้างลิงก์ไปยังหน้าเว็บอื่น ๆ ไฟล์ หรือตำแหน่ง
8. <img>: แทรกรูปภาพเข้าไปในเอกสาร
9. <div>: กำหนดการแบ่งหรือส่วนในเอกสาร HTML
10. <span>: กำหนดส่วนของข้อความหรือเนื้อหาแบบอินไลน์ภายในเอกสาร
11. <ul>: แทนรายการที่ไม่มีลำดับ
12. <ol>: แทนรายการที่มีลำดับ
13. <li>: กำหนดรายการ
14. <table>: สร้างตารางที่มีแถวและคอลัมน์
15. <tr>: กำหนดแถวของตาราง
16. <td>: กำหนดเซลล์ของตาราง
17. <th>: กำหนดเซลล์หัวเรื่องในตาราง
18. `<form>`: สร้างแบบฟอร์ม HTML สำหรับการป้อนข้อมูลจากผู้ใช้
19. <input>: กำหนดองค์ประกอบป้อนข้อมูลภายในแบบฟอร์ม
20. <button>: สร้างปุ่มที่คลิกได้
21. <select>: สร้างรายการแบบเลือกดรอปดาวน์
22. <option>: กำหนดตัวเลือกในรายการเลือก
23. <textarea>: กำหนดตัวควบคุมป้อนข้อความหลายบรรทัด
24. <label>: กำหนดป้ายชื่อสำหรับองค์ประกอบป้อนข้อมูล
25. <br>: แทรกตัวขึ้นบรรทัดใหม่
26. <hr>: แทรกเส้นคั่นหรือเส้นนอน
27. <iframe>: ฝังเอกสารอื่นในเอกสาร HTML ปัจจุบัน
28. <audio>: ฝังเนื้อหาเสียง
29. <video>: ฝังเนื้อหาวิดีโอ
30. <canvas>: กำหนดพื้นที่สำหรับวาดกราฟิกโดยตรง
31. <script>: ฝังหรืออ้างอิงไปยังโค้ด JavaScript ภายนอก
32. <style>: มี CSS (Cascading Style Sheets) สำหรับการกำหนดรูปแบบเอกสาร
33. <meta>: ให้ข้อมูลเมตาเกี่ยวกับเอกสาร HTML
34. <footer>: กำหนดส่วนท้ายสำหรับเอกสารหรือส่วน
35. <header>: กำหนดส่วนหัวสำหรับเอกสารหรือส่วน
36. <nav>: กำหนดส่วนการนำทาง
37. <section>: กำหนดส่วนภายในเอกสาร
38. <aside>: กำหนดเนื้อหานอกเนื้อหาหลัก
39. <blockquote>: กำหนดข้อความอ้างอิงยาว
40. <cite>: ระบุอ้างอิงหรือแหล่งที่มาของ
